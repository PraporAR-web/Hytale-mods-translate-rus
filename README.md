# Hytale — перевод модов

Приложение для перевода модов Hytale на Русский язык. Вся работа ведётся в папке **mods**: автораспаковка, поиск текстов, ручной перевод, сборка.

**Разработчик:** PraporAR (Karnet)

## Инструкция по установке:

 Установка:
 Скачать и расположить HytaleModsTranslator_v1.0.1.exe в любую удобную для вас папку, при запуске приложения, оно создаст папку mods, в которую необходимо разместить модификации для перевода


## Первый способ перевода
  1. Запустите приложение
  2. Выберите желаемый мод и нажмите кнопку "перевести"
  3. Нажмите кнопку "Загрузить строки"
  4. Там где написано "перевод" вручную введите желаемый перевод
  5. Нажмите кнопку "Сохранить и собрать"
  6. в папке mods создастся файл с именем мода с окончанием _rus.zip(_rus.jar) который перемещаете по пути C:\Users\Имя Пользователя\AppData\Roaming\Hytale\UserData\Mods
## Второй способ перевода
  1. Запустите приложение
  2. Выберите желаемый мод и нажмите кнопку "перевести"
  3. Нажмите кнопку "Загрузить строки"
  4. Нажмите кнопку "Перевести все". Пойдет процесс перевода, приложение может зависнуть но процесс будет идти. По завершению процесса откроеться окно в котором вам сообщат о количестве переведенных строк
  5. Нажмите кнопку "Сохранить и собрать"
  6.в папке mods создастся файл с именем мода с окончанием _rus.zip(_rus.jar) который перемещаете по пути C:\Users\Имя Пользователя\AppData\Roaming\Hytale\UserData\Mods

## OTHER

1. **Папка mods** — по умолчанию это `mods` в корне проекта. В ней лежат JAR и ZIP модов. Можно выбрать другую папку («Выбрать…»).

2. **Моды (JAR/ZIP)** — список архивов. Кнопка **«Перевести»**:
   - автоматически распаковывает мод в `mods/.extracted/<имя_мода>/`;
   - находит все строки для перевода (manifest, .lang, .ui, Common/Translations, JSON);
   - открывает окно со списком: **исходный текст | поле для перевода**.

3. В окне перевода:
   - **«Загрузить строки»** — обновить список строк из распакованной папки;
   - заполните столбец «перевод» для нужных строк;
   - **«Перевести все»** — сохраняет переводы в файлы распакованного мода и собирает мод обратно в JAR/ZIP (оригинал при первой сборке копируется в `mods/backups/`).

4. **Распакованные** — моды, которые уже лежат в `mods/.extracted/`. Можно снова открыть перевод (без перезаписи архива) и доредактировать или пересобрать.

## Где ищутся тексты

- **manifest.json** — поля Name, Description, Authors[].Name
- **Server/Languages/**/*.lang** — строки вида `ключ = значение`; перевод пишется в `ru-RU/server.lang`
- **Common/Translations/*.json** — ключ–значение; перевод в `ru_RU.json`
- **.ui** — атрибуты `Text: "..."` и `@Text = "..."`; заменяются в тех же файлах
- **JSON** в Server/ и др. — поля name, description, title, text, displayName, message, lore и т.п.

Переводы записываются в те же файлы внутри распакованной папки, после чего мод собирается обратно в архив.
